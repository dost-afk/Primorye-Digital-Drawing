# Primorye-Digital-Drawing
Primorye Digital Drawing — программа для цифрового рисования, предлагающее уникальный способ изучения культурного и природного наследия Приморского края. В основе проекта лежит концепция преобразования артефактов, представителей флоры и фауны региона в специализированные кисти для создания абстрактных композиций. Каждый цифровой инструмент сопровождается кратким описанием своего прототипа, обеспечивая синтез творческого процесса и образовательной функции.

## О программе

Интерфейс приложения реализован в контрастной темной цветовой схеме с возможностью сворачивания панелей управления для максимального погружения в процесс рисования.
Основной задачей проекта является создание нового формата взаимодействия с культурным наследием через интерактивное цифровое искусство, ориентированного на широкую аудиторию — от студентов и педагогов до ценителей искусства и краеведения.

## Задачи программы

Данная программа поможет погрузиться во флору и фауну Приморского края, узнать больше о территории Дальнего Востока, которая так близка к Азии, но обладает своими аутентичными природными и животными представителями.
