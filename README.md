# Primorye-Digital-Drawing
Primorye Digital Drawing — интерактивное приложение для цифрового рисования, предлагающее уникальный способ изучения культурного и природного наследия Приморского края. В основе проекта лежит концепция преобразования артефактов, представителей флоры и фауны региона в специализированные кисти для создания абстрактных композиций. Каждый цифровой инструмент сопровождается кратким описанием своего прототипа, обеспечивая синтез творческого процесса и образовательной функции.
